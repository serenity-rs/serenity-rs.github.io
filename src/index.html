Serenity, a Rust wrapper for the Discord API

Serenity is a HTTP and gateway wrapper around Discord's API, written in the Rust programming language.

It offers a high-level, easy to use API for creating bots using Discord's official API.

  • Poise

    Poise is a command framework for Serenity.

    It has support for prefix-based commands (e.g. !ping), as well as Discord's application commands, which natively integrate with Discord's client.
